# Bookish Corner

Кроссплатформенное мобильное приложение для управления личной библиотекой электронных и аудиокниг с трекером чтения, заметками, синхронизацией и рекомендациями.

---

## О проекте

Bookish Corner помогает читателям собирать книги в одном месте, отслеживать прогресс чтения, вести заметки и получать персональные рекомендации на основе прочитанного. Приложение работает на iOS и Android, данные синхронизируются через облачный бэкенд.

Ключевые возможности:

- Личная библиотека электронных и аудиокниг
- Трекер чтения с историей и статистикой
- Заметки и цитаты с привязкой к странице
- Синхронизация между устройствами
- Персональные рекомендации

---

## Стек технологий

| Слой | Технологии |
|---|---|
| Клиент | Flutter, Dart, Riverpod |
| Бэкенд | FastAPI, PostgreSQL, RabbitMQ |
| Инфраструктура | Docker, GitHub Actions, Coolify, Traefik |
